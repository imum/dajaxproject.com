Simple dajaxice example

This is the basic example. We'll ask for a message from the server and show it the browser.

ajax.py

from django.utils import simplejson
from dajaxice.decorators import dajaxice_register

@dajaxice_register
def dajaxice_example(request):
    return simplejson.dumps({'message':'Hello from Python!'})

html

<input type="button" onclick="Dajaxice.examples.dajaxice_example(my_callback)" value="Get message from server!">

javascript

function my_callback(data){
    alert(data.message);
}
